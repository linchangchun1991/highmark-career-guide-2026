Part 01

产品矩阵

我们卖的不是课，卖的是结果。17款千里马产品 + 400+ 实习 SKU，覆盖每一类学员在求职路上会遇到的每一个问题。

我们和竞品根本的区别

很多机构卖"背景提升""能力成长"——听起来很好，但学员毕业了还是不知道自己能去哪。海马职加只认一件事：你手里有没有 Offer。

维度市面常见做法海马职加
目标定义背景提升、能力成长以全职 Offer 为唯一交付标准
服务逻辑卖课时、卖体验卖结果——进得去、留得下、发展好
交付标准完成课时数拿到能力范围内最好的名企 Offer
服务方式模块拼凑方向 → 能力 → 投递 → Offer 全链路闭环
留学生求职从来不是靠一次突击能搞定的。它复杂、周期长、容错率低——一个环节掉链子，整个秋招就白搭。我们做的事，是把这条路拆开、配好人、陪着走完。

Part 02

客户画像

见人说人话不是讨好，是效率。你面对的客户不一样，切入的角度就不一样。

A 类
焦虑型家长  ·  转化率最高的一类
核心焦虑花了百万送孩子出去，回来能找到好工作吗？出来和没出来的比，到底有没有竞争力？
决策模式家长主导决策，孩子配合执行
付费动机买的是确定性——花钱买心安，留学是"最后一战"
转化关键案例说话 + 时间线倒逼 + 结果保障
顾问操作要点
  • 先聊大环境——2025年归国留学生突破60万，同比增长10%，竞争实实在在在那里
  • 用时间线倒逼——"孩子明年8月毕业，秋招今年9月就开始了，有效准备时间就这几个月"
  • 找同背景案例对标——不泛泛讲，直接找与对方孩子情况接近的学员，讲他的经历和结果
  • 落在团队保障——"5对1全程跟进，不是上完课就消失"
B 类
迷茫型学生  ·  需要先唤醒，再转化
核心状态不知道自己能做什么，也没意识到问题有多严重，觉得还早
转化路径1v1职业规划体验 → 课时包起步 → 升级全案
操作关键先让他自己看到差距，而不是你告诉他有差距
顾问操作要点
  • 问三个问题让他自己发现差距："你知道你专业对口的岗位有哪些吗？你知道秋招几月份开始吗？你现在有几段实习？"
  • 用免费/低价的1v1职业规划做切入点，让他自己摸到差距
  • 起步签课时包，在服务过程中升级到全案
C 类
紧迫型学生  ·  最好成交的一类，不要绕圈子
核心状态秋招已经开始，投了没回音，或者马上毕业还没Offer，非常焦虑
付费动机极高——愿意为确定性付费，不在乎价格
转化路径直接推千里马全程护航/冲刺拿Offer，快速成交
顾问操作要点
  • 快速建立信任——"你这种情况我们处理过很多，关键是时间窗口，现在来还来得及"
  • 直接展示同背景案例 + 结果时间线，让他看到可能性
  • 主推5对1全配——网申代投 + 1v1面试 + 内推资源全到位
D 类
观望型客户  ·  要用时间养，不能急

低年级或刚出国，知道求职这件事但还没到"着急"的状态。不催、不逼，持续出现——每周推1-2条求职资讯，定期邀请讲座，话术重点："早准备的人不是更焦虑，是更主动。"

核心认知
  • 这类客户要用时间养，不能急。但不养就白白流失
  • 转化路径：内容种草 → 讲座激活 → 1v1体验 → 课时包 → 升级全案
E 类
二次转化客户  ·  你手里最值钱的一批人
关键数据 我们的求职客户中，约40%曾经通过留学机构完成出国申请。你手里的留学老客户，就是求职项目的天然精准池——信任已经建立，只需要延伸话题。
顾问操作要点
  • 在孩子刚到海外时就介入职业规划话题，不要等他们主动问
  • 逻辑是"留学和就业是一体的，出去是为了回来更好"
  • 用你了解的孩子的情况，直接指出求职的薄弱点，拿案例引发共鸣

Part 03

27届求职趋势

8大赛道深度拆解，全球政策红线速查。顾问对赛道了解得越深，客户就越信任你。

先想清楚打哪条路

主线 A
大陆校招（主力首选）

秋招为核心，春招做补位。7月提前批开始，9月是高峰，窗口就这几个月。目标是国内互联网/金融/央国企/制造新能源，竞争激烈但容量大。

主线 B
海外就业（做上限）

英美澳加新港，Graduate Program + 滚动岗位 + 雇主担保路径。海外更看实习经历 + 可交付的项目 + 雇主是否友好。

核心策略 大陆锁底盘，海外做上限，双路径对冲风险——别把鸡蛋放在一个篮子里。

Part 04

秋招节奏

从现在到毕业入职，四个阶段，每个阶段有不同的重点。给客户讲清楚这个时间线，成交率会高很多。

全年四阶段时间线

01
现在 — 2026年5月
确定方向，补齐能力，锁暑期实习

这个阶段最容易被浪费——没有紧迫感，所以拖。但实际上，暑期实习决定了秋招的起点。

必须完成的四件事
简历双语版（中文投递版 + 英文版）
岗位清单（30家，分冲刺/匹配/保底三档）
作品集或项目证明（可展示，能讲故事）
面试素材库（自我介绍 / 项目深挖 / 行为面框架）
02
2026年6月 — 10月
秋招抢跑——这是全年最重要的三个月

7月提前批开始，9月是高峰，10月多数公司收尾。用投递密度 + 笔面系统拿到第一波 Offer。

时间行动
周一筛岗位 + 投递5-10个
周二笔试训练
周三项目深挖 + 模拟面
周四网申补投 + 跟进进度
周五行业研究
周末面试复盘 + 简历迭代

目标：全秋招总投递80-150家，每周10-20家，持续优化而非一次性海投。

03
2027年2月 — 4月
春招翻盘
  • 秋招有面试但没Offer——重点补面试表达和项目深挖
  • 秋招投递少——扩大投递量，调整投递策略
  • 方向选错了——允许换赛道，快速补齐证据
04
2027年5月 — 8月
Offer 选择与入职
  • 多Offer对比表：薪资 / 城市 / 成长空间 / 晋升路径 / 业务方向 / 稳定性
  • 谈薪准备：市场数据锚点 + 竞争性Offer做筹码
  • 入职前补齐：Excel/SQL/写作/汇报/业务基础

Part 06

成交话术

9个异议，每一个都有破解之道。话术是参考，但逻辑才是核心——理解每个异议背后的真实顾虑，才能真正化解。

1
价格太贵了
核心逻辑：贵不贵看ROI，留学花了百万，求职是这笔钱能不能回本的最后一步

我理解，这确实不是小数目。但您算一笔账——留学投了大几十万甚至上百万，如果回来拿的Offer差一个档次，年薪就差10-20万，工作五年就差了50-100万。我们做的不是在锦上添花，是帮您确保这笔留学投资能真正兑现。

而且我们是结果导向的——不是上完课就结束，是拿到Offer才算交付完成。您付的不是课时费，是一个确定性的结果。

2
孩子还早，不着急
核心逻辑：秋招的窗口是有限的。等"意识到着急"的时候，提前批已经结束了

很多家长都这么想，但时间线不会等我们。秋招7月就开始提前批，您孩子明年8月毕业，算下来有效准备时间就这几个月。我们今年2月签约的一个学员，最后拿了8个Offer；9月才来的，1个月也拿到了，但选择面窄很多。准备时间拉长，最直接的变化就是选择权——能去的地方更多，谈薪底气更足。

3
孩子自己准备就行
核心逻辑：留学生DIY被三个问题卡死——信息差、时间差、专业度差

自己准备当然可以，有人成功，但比例很低。留学生面对的不是努不努力的问题，是三个差——人在海外，不知道国内秋招什么节奏；上课 + 论文 + 考试，根本没时间系统投递；第一次经历校招，群面/Case面/结构化面试全靠摸索。我们把这三个问题都解决了——网申代投节省时间，1v1导师填补专业度，内推资源打通信息差。自己准备三个月能做到的事，我们帮他一个月到位。

4
效果能保证吗
核心逻辑：信心建立在数据和案例上，而不是空头承诺上

我不给空头承诺，但我可以给数据——往届学员人均拿到3个以上名企Offer。您说的这种情况我们见过很多，有个和您孩子背景非常接近的学员，0实习起步，4个月拿了6个Offer，我把案例发您看看。

我们是5对1服务，每周都有进度反馈，您随时能看到孩子在做什么——不是签了就消失的那种。
